# 03 · 认证子系统

> 本篇是「解析文档」示例，演示格式。分析对象：`cybertron-api/auth/`。
> 相关源码：`auth/middleware.py`、`auth/token.py`。

---

## 1. 职责概述

负责请求鉴权：解析 Token → 校验签名与有效期 → 注入用户上下文。

## 2. 核心组件

| 组件 | 文件 | 作用 |
|------|------|------|
| `AuthMiddleware` | `auth/middleware.py` | 拦截请求，触发鉴权 |
| `TokenVerifier` | `auth/token.py` | 校验 JWT 签名与过期时间 |

## 3. 执行链路

```
请求 → AuthMiddleware → TokenVerifier.verify() → 注入 user → 业务处理
```
